Welcome to Launchbox

This is a site powered by the Launchbox Demo Service. If you've just installed Launchbox for the first time, seeing this means that everything is working as expected!

This demo has three components that are configured using the routes section of the launch.yaml file:

  • This page is made available by the pages route, defined in the launch.yaml file with: 
    pages:
  route: /
  type: static
  options:
    path: pages
  • Sample Python API endpoint, made available by the api route, defined in the launch.yaml file with: 
    api:
  route: /api
  type: wsgi
  options:
    path: app/api.py
    var: app
  • Sample static JSON endpoint, made available by the assets route, defined in the launch.yaml file with: 
    assets:
  route: /static/data.json
  type: static
  options:
    path: static/data.json
